Quran Audio API

Bienvenue sur l'API Quran Audio. Cette API fournit des fichiers audio du Coran récités par Al Hussary.

Sources Audio

Obtenir les informations sur les sources audio disponibles :

GET /data/audio-sources.json

Exemple de réponse :

{
  "hussary": {
    "baseUrl": "https://VOTRE_NOM_UTILISATEUR.github.io/quran-audio/audio/hussary",
    "format": "mp3"
  }
}

Fichiers Audio

Pour accéder à un fichier audio spécifique, utilisez l'URL suivante :

GET /audio/hussary/{surah_number}/{filename}

Exemple :

GET /audio/hussary/001/001_Al-Fatiha.mp3

Pour utiliser cette API dans votre application, construisez les URLs en utilisant le baseUrl fourni dans audio-sources.json.
